# Vyral

**Vyral** is an InfoFi (Information Finance) platform built on **Base**. It enables campaign creation, creator rewards, and community engagement through NFTs and on-chain interactions.

---

## Purpose

Vyral is designed to:

- **Run InfoFi campaigns** — Create and manage campaigns that reward participation and mindshare.
- **Reward creators** — Distribute rewards to creators via dedicated NFT collections and in-app mechanics.
- **Connect Twitter identity to Web3** — Users sign in with Twitter and get an embedded wallet; no seed phrases required.
- **Provide a lightweight Web3 experience** — Marketplace, pack shop, and referrals all within one app on Base.

The platform targets communities and creators who want to run tokenized campaigns and reward their audience without heavy crypto onboarding.

---

## Skills & Capabilities

| Area | Description |
|------|-------------|
| **Authentication** | Twitter-only login via Privy; embedded in-app wallets. |
| **Creator NFTs** | Per-creator NFT collections (e.g. 100 supply) for rewards and collectibles. |
| **Campaigns** | InfoFi campaigns with mindshare and reward logic. |
| **Marketplace** | Buy and sell NFTs on the platform. |
| **Pack Shop** | Randomized card/pack opening for collectibles. |
| **Referrals** | User referral system to grow the community. |
| **Admin** | Full admin panel for campaigns, creators, packs, and users. |

See [FEATURES.md](./FEATURES.md) for a concise list of features and tech stack.

## Tech Overview

- **Frontend:** Next.js 14 (App Router), React 18, TypeScript, Tailwind CSS  
- **Backend:** Next.js API Routes, Prisma ORM  
- **Database:** PostgreSQL  
- **Auth:** Privy (Twitter OAuth)  
- **Chain:** Base (Ethereum L2)
